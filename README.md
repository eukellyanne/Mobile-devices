# Programação de Dispositivos Móveis

## Descrição do Projeto

Desenvolver um site que funcione bem, seja fácil de usar e se adapte a diferentes tamanhos de tela, como computadores e celulares, garantindo uma boa experiência para todos os usuários.


## Estrutura da Branch

- `main`: Branch principal de desenvolvimento, onde todas as alterações e novas funcionalidades serão implementadas e testadas.

## Documentação das Entregas 

**Entrega 1**

- Exercicio 1: O componente Exercicio1 recebe uma propriedade chamada nome e exibe uma saudação simples com o texto “Olá, {nome}”. O conteúdo está centralizado na tela, com uso das propriedades justifyContent e alignItems em conjunto com padding para espaçamento.

![Exercicio1](https://github.com/user-attachments/assets/d54a8850-21b0-4956-a5af-fb1dabcbe64f)

- Exercicio 2 e 3 : O componente Exercicio2 utiliza o hook useState para controlar um contador numérico. Criei dois botões: um para incrementar e outro para decrementar o valor do contador. O layout também está centralizado na tela com padding, justifyContent e alignItems.

![Exercicio2e3](https://github.com/user-attachments/assets/a86c3328-1862-4d98-a2dc-de7e57e78cab)

- Exercicio 4: Nesse exercicio, foi solicitada uma lista de times, essa lista foi definida em um array, e exibida dinamicamente com o uso do método map(). Cada clube é exibido em uma nova linha, com o auxílio da função de mapeamento.

![Exercicio4](https://github.com/user-attachments/assets/1a49f38e-b6f5-45fb-8e53-ce24ef340c15)


**Entrega 2**

-Exercicio 1: Utilizei um contêiner principal para estruturar a tela, integrando o cabeçalho, o conteúdo central e o rodapé. Esse contêiner foi configurado para ocupar todo o espaço disponível, mantendo os elementos centralizados. Dessa forma, o cabeçalho permanece na parte superior, o conteúdo no centro e o rodapé na parte inferior da tela.
